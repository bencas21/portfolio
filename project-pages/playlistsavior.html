Project Details / Background
OrgReport is a web application designed for anonymous and signed-in reporting of misconduct within organizations at the University of Virginia. Developed using Django for the backend and Bootstrap for the frontend, OrgReport provides a user-friendly interface for students to report issues related to clubs, organizations, and Greek life. At this time, this app is just a prototype and there are no real entries of misconduct
he application supports both anonymous reporting and signed-in features via Googleâ€™s OAuth API for secure authentication. All reports and static files are hosted on Heroku, with a PostgreSQL database managing report data and AWS S3 storing static files and form attachments. An admin view is integrated to allow administrators to review and manage reports, mark them as finished after providing feedback. This comprehensive solution aims to provide a secure and efficient way for the UVA community to address and resolve issues.
